# GetUndoCount

## Description

Get number of undo action which is capable of running

## Syntax

GetUndoCount()

## Inputs

1. NONE

No arguments

## Return Code

_Python Output_

- int

Number of undo action

## Sample Code

_Input:_

```python
undo = JPT.GetUndoCount()
print(str(undo))
```

_Output:_

\> `2`
