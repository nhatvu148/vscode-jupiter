# GetProgramPath

## Description

Get application installation directory

## Syntax

GetProgramPath()

## Inputs

1. None

No input value

## Return Code

_Python Output_

- String

Program Path

## Sample Code

_Input:_

```python
path=JPT.GetProgramPath()
print(str(path))
```

_Output:_

\> `C:\Program Files\TechnoStar\Jupiter-Pre_4.1\`
