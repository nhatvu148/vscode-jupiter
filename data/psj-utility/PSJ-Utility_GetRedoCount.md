# GetRedoCount

## Description

Get number of redo action which is capable of running

## Syntax

GetRedoCount()

## Inputs

1. NONE

No arguments

## Return Code

_Python Output_

- int

Number of redo action

## Sample Code

_Input:_

```python
redo = JPT.GetRedoCount()
print(str(redo))
```

_Output:_

\> `2`
