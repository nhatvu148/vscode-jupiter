# GetJPTTempPath

## Description

Get temporary document path

## Syntax

GetJPTTempPath()

## Inputs

1. None

No input value

## Return Code

_Python Output_

- String

Temp Document Path

## Sample Code

_Input:_

```python
path=JPT.GetJPTTempPath()
print(str(path))
```

_Output:_

\> `C:\Users\TECHNOSTAR\AppData\Local\Temp\TechnoStar\`
