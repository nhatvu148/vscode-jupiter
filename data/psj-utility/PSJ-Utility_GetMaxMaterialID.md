# GetMaxMaterialID

## Description

Get maximum id of material inside a document.

## Syntax

GetMaxMaterialID()

## Inputs

1. None

No input value

## Return Code

_Python Output_

- int

Material ID

## Sample Code

_Input:_

```python
id = JPT.GetMaxMaterialID()
print(str(id))
```

_Output:_

\> `3`
