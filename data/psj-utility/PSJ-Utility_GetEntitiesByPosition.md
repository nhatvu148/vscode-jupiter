# GetEntitiesByPosition

## Description

Get list of entity object by positionn

## Inputs

1. AssociateType enum type (JPT.AssociateType)

2. x (double)

3. y (double)

4. z (double)

## Return Code

list of Entity object (EntityVector)

## Sample Code

```python
JPT.GetEntitiesByPosition(JPT.AssociateType.AS_BODY, 1, 2, 3)
```
