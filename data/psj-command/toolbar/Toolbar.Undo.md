# Toolbar.Undo

## Description

Undo

## Syntax

```python
Toolbar.Undo(iCntUndo=1)
```

Macro: {ref}`Macro-Toolbar-Undo`

Ribbon: {menuselection}`Toolbar --> Undo`

## Inputs

**`iCntUndo`**
: An _Integer_ specifying the cnt undo. The default value is 1.

## Return Code

An _String_ of 1 if successed, or 0 if failed.

## Sample Code

```python
Toolbar.Undo(iCntUndo=1)
```
