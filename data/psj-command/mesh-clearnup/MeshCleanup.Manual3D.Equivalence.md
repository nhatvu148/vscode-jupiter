# MeshCleanup.Manual3D.Equivalence

## Description

## Syntax

```python
MeshCleanup.Manual3D.Equivalence(crlNode=[], iMergeTowards=0)
```

Macro: {ref}`Macro-MeshCleanup-CleanupEquivalence`

Ribbon: {menuselection}`MeshCleanup --> Manual3D --> Equivalence`

## Inputs

**`crlNode`**
: A _Cursor List_ specifying the node. The default value is [].

**`iMergeTowards`**
: An _Integer_ specifying the merge towards. The default value is 0.

## Return Code

An _String_ of 1 if successed, or 0 if failed.

## Sample Code

```python
MeshCleanup.Manual3D.Equivalence(crlNode=[], iMergeTowards=0)
```
